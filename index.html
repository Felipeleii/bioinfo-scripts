Sobre o Projeto

O Bioinfo Scripts v3.0 é uma coleção unificada de ferramentas e scripts para análise de genomas bacterianos, com foco especial em dados de sequenciamento Illumina.

Características Principais:

  • Interface web intuitiva e responsiva
  • Processamento de arquivos FASTA/FASTQ no navegador
  • Integração com pipelines populares (Bactopia, Prokka, SPAdes)
  • Scripts otimizados para análise de WGS bacteriano
  • Documentação completa e exemplos práticos

Instalação e Uso:

# Clone o repositório git clone https://github.com/Felipeleii/bioinfo-scripts.git cd bioinfo-scripts # Instale as dependências conda env create -f environment.yml conda activate bioinfo-scripts # Execute os scripts ./scripts/run_analysis.sh
