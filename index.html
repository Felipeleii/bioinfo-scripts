Sobre o Projeto

O Bioinfo Scripts v3.0 é uma coleção unificada de ferramentas e scripts para análise de genomas bacterianos, com foco especial em dados de sequenciamento Illumina.

Características Principais

  • Interface gráfica intuitiva para bioinformáticos e microbiologistas
  • Processamento em lote para análises de alto rendimento
  • Integração com pipelines populares como Bactopia e Prokka
  • Visualização de resultados diretamente no navegador
