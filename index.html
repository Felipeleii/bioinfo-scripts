Bioinfo Scripts v3.0

Ferramentas avançadas de bioinformática para análise de genomas bacterianos e sequenciamento Illumina

Ferramentas Principais

Gerador de Script SRA

Crie scripts Bash para baixar e processar dados do SRA em lote

Gerador MultiFASTA

Crie arquivos multiFASTA com rastreamento automático de origem das sequências

Bactopia Prepare GUI

Interface gráfica para preparação de dados do pipeline Bactopia

FASTQ Stats Calculator

Calcule estatísticas básicas de arquivos FASTQ (reads, qualidade, GC%)

Primer Designer

Desenhe primers para PCR com análise de Tm e estruturas secundárias

Genome Statistics

Calcule N50, tamanho do genoma, número de contigs e outras métricas

MLST Analyzer

Análise de Multi-Locus Sequence Typing para tipagem bacteriana

Gerador de Script SRA

Cole sua lista de amostras (SRR e ID), configure as opções e clique em "Gerar Script" para criar um script Bash pronto para uso no Linux.

                    Sobre o Projeto
                

                
O Bioinfo Scripts v3.0 é uma coleção unificada de ferramentas e scripts para análise de genomas bacterianos, com foco especial em dados de sequenciamento Illumina.

                
                
Scripts Disponíveis para Download:

                

                    
                        
                        
illumina_qc_pipeline.sh

                        
Pipeline completo de QC para dados Illumina

                    
                    
                        
                        
bacterial_assembly.py

                        
Automatização de montagem com SPAdes/Unicycler

                    
                    
                        
                        
multifasta_generator.py

                        
Versão CLI do gerador multiFASTA

                    
                    
                        
                        
bactopia_helper.sh

                        
Scripts auxiliares para Bactopia
